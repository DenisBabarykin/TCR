Web-приложение для анализа репертуаров Т-клеточных рецепторов

Индивидуальные распределения:

Распределение длин

Построение гистограммы распределения длин нуклеотидных последовательностей Т-клеточных рецепторов в репертуаре выбранного человека из базы.

Распределение V-сегментов

Построение гистограммы количественного распределения каждого вида V-сегмента в репертуаре выбранного человека из базы.

Распределение D-сегментов

Построение гистограммы количественного распределения каждого вида D-сегмента в репертуаре выбранного человека из базы.

Распределение J-сегментов

Построение гистограммы количественного распределения каждого вида J-сегмента в репертуаре выбранного человека из базы.

Общие распределения:

Распределение количества клонотипов

Построение heatmap с количеством общих клонотипов нуклеотидных последовательностей между всеми репертуарами в базе.

Распределение коэффициентов клонотипов

Построение heatmap с коэффициентами количества общих клонотипов нуклеотидных последовательностей между всеми репертуарами в базе. Коэффициенты рассчитываются делением количества общих клонотипов на произведение объёмов рассмотренных репертуаров и домножением на константу для нормализации результатов.
