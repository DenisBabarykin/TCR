Web-приложение для анализа репертуаров Т-клеточных рецепторов

Индивидуальные распределения:

Распределение длин

Построение гистограммы распределения длин нуклеотидных последовательностей Т-клеточных рецепторов в репертуаре выбранного человека из базы.

Распределение V-сегментов

Построение гистограммы количественного распределения каждого вида V-сегмента в репертуаре выбранного человека из базы.

Общие распределения:

Распределение клонов

Построение heatmap с количествами общих клонов нуклеотидных последовательностей между всеми репертуарами в базе.
